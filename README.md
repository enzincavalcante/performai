# PerformAI

Plataforma de inteligência e performance comercial para acompanhar pessoas, metas, ranking, equipes, comunicação e reconhecimento.

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

O deploy para GitHub Pages é executado automaticamente pela workflow `deploy-pages.yml` após atualizações na branch `main`.

## Acesso de demonstração

- Usuário: `Cavalcante`
- Senha: definida para o ambiente de demonstração solicitado

O acesso de demonstração é local ao front-end. Usuários e dados reais continuam dependendo da configuração Supabase do projeto.
